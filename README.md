# QDB paper from master's thesis

This workspace uses the official VLDB workshop template linked from the QDB
2026 submission page. The template is kept in `vldb-workshop-style-master/`.

Working sources:

- `vldb-workshop-style-master/paper-en.tex`: English version.
- `vldb-workshop-style-master/paper-es.tex`: Spanish version.
- `vldb-workshop-style-master/references.bib`: shared bibliography.

Current QDB constraints checked on 2026-05-25:

- Submit through the QDB 2026 CMT site.
- Use the VLDB workshop format linked by QDB.
- Maximum length: 6 pages, including figures, tables, and references.

Incremental workflow:

1. Add the thesis source material to this workspace.
2. Extract the core thematic axes, contribution, methods, evidence, and related
   work.
3. Draft the English paper first, because that is the likely submission
   language.
4. Mirror the accepted structure into the Spanish version.
5. Compile both PDFs and check page count, figures, references, and formatting.
